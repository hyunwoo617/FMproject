# 📻 FM Radio - 관악FM 방송 자동화 시스템 v3

현대적인 인터페이스와 클라우드 보안을 결합한 관악FM 전용 방송 자동화 대시보드입니다.

## 🚀 시작하기 (사용자 가이드)

운영진 및 사용자는 아래 단계에 따라 시스템을 이용할 수 있습니다.

### 1. 시스템 접속
*   **클라우드 주소**: `https://fmproject-80700.web.app` (또는 배포된 URL)
*   **로그인**: 관리자가 생성한 Firebase 계정(이메일/비밀번호)으로 로그인합니다.

### 2. 주요 기능 활용
1.  **메인 (대시보드)**
    *   현재 방송 현황, 분야별 비율, 결재 대기 건수를 한눈에 확인합니다.
    *   프로그램 신규 등록 및 SB Time 관리가 가능합니다.
2.  **승인 프로세스**
    *   결재가 필요한 문서를 제출하거나, 관리자가 승인/반려 처리를 수행합니다.
3.  **편성 및 보고**
    *   주간 편성표를 확인하고, 방통위 제출용 **월간 방송실시결과보고서**를 생성합니다.
    *   본방/재방 비율 분석 및 공익광고 가중치 현황을 파악합니다.
4.  **폴더 저장소**
    *   발행된 모든 보고서와 편성표 데이터를 안전하게 보관하고 필요시 다운로드합니다.
5.  **관리자 설정**
    *   Excel 데이터를 일괄 업로드하거나 시스템 환경 설정을 관리합니다.

## 🛠️ 개발자/관리자 가이드

### 기술 스택
*   **Frontend**: HTML5, Vanilla CSS3 (Glassmorphism), JavaScript
*   **Backend/Cloud**: Firebase (Authentication, Hosting)
*   **Library**: SheetJS (Excel 처리), Google Fonts (Outfit, Inter)

### 로컬 실행 방법
1.  저장소를 클론합니다: `git clone [repository-url]`
2.  `index.html` 파일을 브라우저로 엽니다.

### 클라우드 배포 방법
수정 사항을 서버에 반영하려면 터미널에서 아래 명령어를 실행합니다.
```bash
firebase deploy
```
